# Python Automation Portfolio

This repository contains practical Python automation scripts for real-world tasks.

## 🔧 Skills Demonstrated
- File Automation
- Data Cleaning (CSV)
- Bulk File Management
- Backup Automation
- Productivity Tools

## 📁 Projects Included

### 1. File Organizer
- Automatically organizes files by extension
- Uses pathlib for clean file handling

### 2. Bulk File Renamer
- Renames multiple files with custom prefix and numbering
- Useful for invoices, images, documents

### 3. CSV Cleaner
- Cleans messy CSV files
- Removes empty rows and trims spaces

### 4. Folder Backup Tool
- Creates automatic timestamp backups of folders
- Safe and structured backup system

## 💻 Tech Stack
- Python 3
- Standard Library (pathlib, shutil, csv)
- Automation-focused scripting

## 🎯 هدف (Goal)
To build efficient automation tools that save time and improve productivity.
